# Boolean Values: - It Can Be Either True Or False. - It Is Used In Conditional Statements, loops & Comparisons To Control The Flow Of Execution Based On Conditions. - Declaring & Assigning Boolean Values: let isTrue = true; let isFalse = false; # Comparision Operators & Boolean Result: 1. Equality(==): - Check If Two Values Are Equal & Return True If They Are, False Otherwise. Eg: O/p: Equility Checked ✅ 2. Inequality(!=): - Check If Two Values Are Not Equal & Returns True If They Are Not, False Otherwise. Eg: O/p: Inequility Checked ✅ 3. Strict Equality(===): - Check If Two Values Are Equal In Both Value & Data Type & Returns True If They Are, False Otherwise. Eg: O/p: Strictly Equility Checked ✅ 4. Strict Inequality(!==): - Checks If Two Values Are Not Equal In Either Value Or Data Type & Returns True If They Are Not, False Otherwise. Eg: O/p: Strictly Inequility Checked ✅ 5. Greater Then(>): - Checks The Value On Left Side Is Greater Then Value Of Right Side & Returns True, Otherwise False. Eg: O/p: Greater Then Condition Checked ✅ 6. Less Then(<): - Checks The Value On Left Side Is Less Then Value Of Right Side & Retuen True, Otherwise False. Eg: O/p: Less Then Condition Checked ✅
