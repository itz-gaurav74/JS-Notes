# Array - Ordered Collection Of Items. - This Items Stored Inside Of SquareBrackets []. - We Can Store All Data Types Like string,number,null,booleans... - The Type Of An Array Is `Object`. - Array Are Refrence Type. - Array Are Mutable / Change The Original Array. Syntax: // Create A Array let arr = []; Example: O/p: [1,2,3,4,5] Example: O/p: ['Hello', 'World', 'Sky', 'Dev'] Example: O/p: false Conclusion: [] === [] returns false because === checks for reference equality when comparing objects (including arrays), and each [] creates a distinct array object with a different reference in memory. Therefore, they are not strictly equal according to the === operator.
