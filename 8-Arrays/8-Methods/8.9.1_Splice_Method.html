# Splice Method: - It Modifies An Array (Adds, Removes Or Replaces Elements). - It Change The Original Array. Syntax: arr.splice(start, delete, insert); Example: // Remove The Ele In Index 4 O/p: (6) [1, 4, 3, 2, 7, 8] Example 2: // Insert An Ele 5 At The Index Of 4 O/p: (8) [1, 2, 3, 4, 5, 6, 7, 8]
