# ForEach() - It Executes A Provoded Function Once For Each Array Element. - It Dosen't Return A New Array; It Just Iterate Through Array. - It Is Used When You Want To Perform An Action Or Operation On Each Element Of The Array Without Creating A New Array. - It Takes A CallBack Function As Input. Syntax: Example: // Using ForEach On Array Ele O/p: 4 8 12 16 Example 2: // Using ForEach Taking As CallBack Function O/p: User Index : 0, & Salary : 25K User Index : 1, & Salary : 55K User Index : 2, & Salary : 24K User Index : 3, & Salary : 45K Example 3: // Using ForEach Taking Function As Callback O/p: User Id : 0's Salary Is 25K User Id : 1's Salary Is 55K User Id : 2's Salary Is 24K User Id : 3's Salary Is 45K Example 4: // Using ForEach Create An Array And Multiply Each Ele By 2. O/p: At Index 0 Number 2 * 2 = 4 At Index 1 Number 4 * 2 = 8 At Index 2 Number 6 * 2 = 12 At Index 3 Number 8 * 2 = 16 Example 5: // Using ForEach Create A Array Containing Multiple Students Data Like FirstName, Age, Std & Log User's FirstName In Console ? O/p: -----Using ForEach()----- Student UserName : Harry Student UserName : Dev Student UserName : Gaurav Student UserName : Radha -----Using For Of Loop----- User's Name Is Harry User's Name Is Dev User's Name Is Gaurav User's Name Is Radha
