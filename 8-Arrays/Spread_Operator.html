# Spread Operator In Arrays. - In ES6 Introduced A Special Operator `...` Called Spread. - When Spread Operator Is Placed In Front Of An Array, It Splits The Array Into Individuals Values. Example: // Without Spread Opr O/p: 6 Example: O/p: (7) [1, 2, 3, 4, 5, 6, 4] (6) ['a', 'b', 'c', 'd', 'e', 'f']
