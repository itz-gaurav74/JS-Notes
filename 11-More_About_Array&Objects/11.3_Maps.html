# Maps: 1. Key-Value Pairs: - A Map Holds Key-Value Pairs, Where The Key Can Be Any DataType. 2. Original Insertion Order: - This means the first key-value pair added to the Map will be stored first, the second added will be stored second, and so on. 3. Key Data Types: - Keys in a Map can be of any data type, such as numbers, strings, objects, or even other Maps. 4. Type Of Maps Are Objects: O/p: object
