# Maps: 1. Key-Value Pairs: - A Map Holds Key-Value Pairs, Where The Key Can Be Any DataType. 2. Original Insertion Order: - This means the first key-value pair added to the Map will be stored first, the second added will be stored second, and so on. 3. Key Data Types: - Keys in a Map can be of any data type, such as numbers, strings, objects, or even other Maps. 4. Type Of Maps Are Objects: O/p: object # How to Create a Map: - Passing an Array to `new Map()`. - Create a Map and use `Map.set()`. 1. The new Map() Method: Example: O/p: Map(3) {'Apple' => 500, 'Banana' => 300, 'Cherry' => 200} 2. The set() Method: - We Can Add Elements To A Map With The `set()` Method. - Also We Use `set()` Mathod To Chnage The Existing Map Values. Example: O/p: Fruits Stock : Map(0) {size: 0} Fruits Stock Updated: Map(3) {'Apple' => 500, 'Banana' => 300, 'Cherry' => 200} Fruits Stock Modifyed: Map(3) {'Apple' => 500, 'Banana' => 300, 'Cherry' => 400} 3. The get() Method: - The `get()` Method Gets the Value Of A Key In A Map. Example: O/p: 500
