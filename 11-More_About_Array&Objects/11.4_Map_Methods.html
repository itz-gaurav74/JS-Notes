# Map Methods: - Map is a collection of key-value pairs where keys can be any data-type. # How to Create a Map: - Passing an Array to `new Map()`. - Create a Map and use `Map.set()`. 1. The new Map() Method: Example: O/p: Map(3) {'Apple' => 500, 'Banana' => 300, 'Cherry' => 200} 2. The set() Method: - We Can Add Elements To A Map With The `set()` Method. - Also We Use `set()` Mathod To Chnage The Existing Map Values. Example: O/p: Fruits Stock : Map(0) {size: 0} Fruits Stock Updated: Map(3) {'Apple' => 500, 'Banana' => 300, 'Cherry' => 200} Fruits Stock Modifyed: Map(3) {'Apple' => 500, 'Banana' => 300, 'Cherry' => 400} 3. The get() Method: - The `get()` Method Gets the Value Of A Key In A Map. Example: O/p: The Value Of Apple : 500 4. Checking For Existance: - The `has()` method returns true if a key exist in Map. Example: O/p: The brand is exist in map : true The age is exist in map : false 5. Map Size: - The size property returns the number of ele in map. Example: O/p: The Size Of FavCar : 2 6. Clearing a Map: - The clear() method removes all ele from a map.
