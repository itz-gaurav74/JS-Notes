The Pkg Of Dev Is 25LPA In TCS Company # How to Use the 'Apply' Method: - It Is Similar To 'call', But Insted Of Passing Arguments Individually, You Pass Them As An Array. Ex: O/p: Hii Dev, Your EmpCode T246 And EmpPkg 28LPA Is Correct. # How to Use the 'Bind' Method: - Bind Method Is A function That Creates A New Function With A Specified `this` Value And Optionally, A Set Of Arguments. - It Allows You To Control The Context In Which A Function Is Called. Syntax: function.bind(thisArgs, arg1[, arg2[, ...]]); Parameters: 1. thisArgs - The Value To Use As `this` When The New Function Is Called. Example: O/p: 42 When to Use bind: - Event Listeners: Ensure that this inside a callback refers to the object you want. - Function Currying: Create new functions with some arguments preset.
