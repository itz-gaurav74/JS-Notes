# Optional Chaining: - It Allows Us To Safely Access The Properties Or The Call Functions On Potentially Null Or Undefined Values Without Giving Any Runtime Errors. Syntax: obj?.prop obj?.[expr] arr?.[index] func?.(args) Example: O/p: Harry 21 Undefined Explanation: - The Optional Chaining (?.). - It Checks The data Is Exist Or Not If Exists Then Log The Value Otherwise Show Undefined.
