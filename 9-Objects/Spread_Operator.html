# Spread Operator: - Spread Operator With Objects, It Allows Us To Create A New Object By Copying Properties From An Existing Object Or Merging Ptoperties From Multiple Objects. Example: O/p: {key1: 'val1', key2: 'val2', key3: 'val3', key4: 'val4'} Example 2 : O/p: {key1: 'valUnique', key2: 'val2', key3: 'val3', key4: 'val4'} {key3: 'val3', key4: 'val4', key1: 'val1', key2: 'val2'} Example 3: O/p: {0: 'A', 1: 'B', 2: 'C', 3: 'D'} 1. Shallow Copy Of An Object: Example: O/p: {a: 1, b: 2}
