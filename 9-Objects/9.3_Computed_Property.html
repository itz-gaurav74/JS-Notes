# Computed Property. - ES6 Allous Us To Use An Expression In Brackets []. - Then Use The Result Of The Expression As Property Name Of Object. Example: O/p: 3 -- Here `[propName]` is A Computed Property Of Obj Object. -- When You Access `c` Property Of `obj` Object, Js Evalutes `propName` & Returns The Property's Value. Example: O/p: {objKey1: 'objVal1', objKey2: 'objVal2'}
